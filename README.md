## 🎯 학습 방향

이 저장소는 단순한 코드 실습보다 **딥러닝의 수학적·이론적 원리**를 깊이 이해하는 데 목적을 두고 작성되었습니다.

- 각 챕터의 핵심 개념과 수식을 **직접 설명하는 방식**으로 노트 작성
- 코드보다 **"왜 이렇게 동작하는가"** 에 집중
- 수식 유도 과정, 그래프 해석, 직관적 이해를 중심으로 정리
- Colab 환경에서 마크다운 셀과 코드 셀을 혼합하여 서술형으로 구성

> ⚠️ **참고**: 이론 학습에 집중한 특성상, 일부 챕터에서는 **책의 실습 코드 일부가 생략**되어 있습니다.  
> 완전한 구현 코드가 필요하다면 [원저자 공식 GitHub](https://github.com/WegraLee/deep-learning-from-scratch)를 참고하세요.

## 📝 챕터별 학습 내용 요약

### 1장 · 헬로 파이썬
- NumPy 배열 연산, 브로드캐스팅 원리
- Matplotlib을 이용한 데이터 시각화 기초

### 2장 · 퍼셉트론
- 퍼셉트론의 구조와 동작 원리 (AND, OR, NAND, XOR 게이트)
- 단층 퍼셉트론의 한계와 다층 퍼셉트론의 필요성
- ⚠️ *XOR 게이트 구현 코드 일부 생략*

### 3장 · 신경망
- 활성화 함수(시그모이드, ReLU, 항등함수, 소프트맥스)의 수학적 정의 및 특성
- 행렬 곱을 이용한 순전파(forward propagation) 흐름 이해
- ⚠️ *MNIST 추론 구현 코드 생략 (데이터 로딩 부분)*

### 4장 · 신경망 학습
- 손실 함수(MSE, 교차 엔트로피 오차)의 의미와 수식 유도
- 경사 하강법과 수치 미분의 원리
- 미니배치 학습 방식의 이론적 배경
- ⚠️ *2층 신경망 클래스 전체 구현 코드 생략*

### 5장 · 오차역전파법
- 계산 그래프와 연쇄 법칙(chain rule)을 이용한 역전파 개념
- Affine 계층, Softmax-with-Loss 계층의 수식 유도
- ReLU, Sigmoid 계층의 역전파 도출 과정
- ⚠️ *각 계층의 파이썬 클래스 구현 코드 일부 생략*

### 6장 · 학습 관련 기술들
- 가중치 초기값(Xavier, He 초기화)의 원리와 중요성
- 최적화 알고리즘 비교: SGD → Momentum → AdaGrad → Adam
- 배치 정규화(Batch Normalization)의 동작 원리
- 드롭아웃(Dropout) 및 가중치 감쇠를 이용한 오버피팅 방지
- ⚠️ *하이퍼파라미터 튜닝 실험 코드 생략*
